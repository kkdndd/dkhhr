# 챗봇 사용 및 편집 가이드

## 챗봇이란?

사이트 우측 하단의 💬 버튼을 누르면 열리는 인사·복지 도우미입니다.
사용자가 키워드로 질문하면 미리 정의된 Q&A 데이터에서 가장 관련성 높은 답변을 찾아 보여줍니다.

---

## 설치 후 실행

이미 스타터 템플릿에 챗봇이 포함되어 있습니다. 아래 명령으로 실행하면 됩니다.

```
npm install
npm run dev
```

브라우저(`http://localhost:5173`)에서 우측 하단에 💬 버튼이 보이면 정상 동작입니다.

---

## 챗봇 동작 원리

1. 사용자가 입력한 문장에서 공백을 제거한 문자열을 만듭니다.
2. `faq-data.ts` 파일에 정의된 모든 항목의 `keywords` 와 비교합니다.
3. **가장 많이 일치한 키워드를 가진 항목**의 답변을 보여줍니다.
4. 일치하는 항목이 없으면 FAQ 페이지로 안내합니다.

> 단순한 키워드 매칭이므로 AI처럼 의미를 이해하지는 않지만, 잘 만든 키워드 세트로 충분히 실용적입니다.

## Q&A 추가/수정하는 방법

### 파일 위치
```
docs/.vitepress/theme/faq-data.ts
```

### 새 항목 추가 예시

휴가 이월 정책에 대한 Q&A를 추가하고 싶다면:

```typescript
// faq-data.ts 의 배열 안에 추가
{
  keywords: ['이월', '연차이월', '휴가이월'],
  answer: '미사용 연차는 다음 해로 이월되지 않으며, 연차 수당으로 정산됩니다.',
  link: '/welfare/leave'  // 선택사항
},
```

### 키워드 작성 팁

- **띄어쓰기 없이** 작성: `연차이월` (O), `연차 이월` (X)
- **다양한 표현을 모두** 포함: `의료비`, `병원비`, `진료비`
- **너무 짧거나 흔한 단어는 피하기**: `다`, `있나요` 같은 단어는 매칭이 너무 광범위해짐
- **여러 키워드 중 하나만 일치해도** 매칭되지만, 많이 일치할수록 우선순위가 높아짐

### 답변 작성 팁

- **간결하게 1~3문장**으로 작성 (챗봇 말풍선이라 길면 가독성 떨어짐)
- **숫자나 일수**가 있으면 명확히 표기: `5일`, `200만원`
- 자세한 설명이 필요하면 `link` 필드를 추가해서 해당 페이지로 안내

### 답변 우선순위 조정하기

같은 키워드가 여러 항목에 있을 때, **더 많은 키워드와 매칭되는 항목**이 우선됩니다.
구체적인 항목을 먼저 매칭시키려면, 더 많은(혹은 더 구체적인) 키워드를 넣어주세요.

예시:
```typescript
// "연차 신청 어떻게 해요?" 라는 질문에 두 번째 항목이 매칭되도록
{
  keywords: ['연차'],
  answer: '연차는 15일 부여됩니다...',
},
{
  keywords: ['연차', '신청', '어떻게'],
  answer: '연차는 인트라넷 근태관리에서 신청하세요...',
},
```

## 빠른 질문 버튼 수정

채팅창 하단의 빠른 질문 버튼을 수정하려면:

**파일**: `docs/.vitepress/theme/ChatBot.vue`

```typescript
const quickQuestions = [
  '연차 며칠?',
  '경조금',
  '의료비 청구',
  '평가 등급',
  '승진 요건'
]
```

이 배열의 항목을 자유롭게 수정하면 됩니다. 4~6개 정도가 적당합니다.

## 챗봇 색상 변경

채팅창과 버튼 색상은 VitePress의 브랜드 색을 따릅니다.
변경하려면 `docs/.vitepress/theme/style.css` 파일을 만들고 다음을 추가하세요.

```css
:root {
  --vp-c-brand-1: #1e88e5;  /* 원하는 색 */
  --vp-c-brand-2: #1565c0;
}
```

그리고 `theme/index.ts` 상단에 `import './style.css'` 한 줄을 추가하세요.

---

## 챗봇 위치 변경

기본은 우측 하단입니다. `ChatBot.vue` 의 `<style>` 부분에서 변경 가능:

```css
.chatbot-fab {
  bottom: 24px;  /* 아래에서 떨어진 거리 */
  right: 24px;   /* 오른쪽에서 떨어진 거리 */
}
```

좌측 하단으로 옮기려면 `right: 24px` → `left: 24px` 로 바꾸세요.

## 자주 묻는 관리자 질문

### Q. AI 챗봇으로 업그레이드하고 싶어요
나중에 Claude API나 OpenAI API를 연동해 AI 챗봇으로 바꾸려면:
- Netlify Functions(serverless)로 API 호출
- 본 가이드의 키워드 매칭 부분만 API 호출로 교체
- API 비용은 월 1~3만원 수준 (사용량에 따라 다름)

원하시면 별도 가이드를 만들어드릴게요.

### Q. 챗봇 대화 내용을 저장할 수 있나요?
현재는 페이지 새로고침 시 대화가 초기화됩니다.
저장이 필요하면 `localStorage`에 대화 내역을 저장하도록 확장할 수 있습니다.

### Q. 키워드를 자주 업데이트해도 되나요?
네, `faq-data.ts` 파일만 수정하고 GitHub에 푸시하면 자동 배포됩니다.
경영전략팀이 직원들의 실제 질문을 모아서 정기적으로 키워드를 보강하면 챗봇 정확도가 점점 올라갑니다.

### Q. 챗봇이 답변을 못 찾을 때 로그를 보고 싶어요
"답을 찾지 못했어요" 메시지가 나오는 빈도를 추적하려면:
- Google Analytics 또는 Plausible 같은 분석 도구 연동
- 또는 챗봇 컴포넌트에 fallback 발생 시 경영전략팀 이메일로 알림 보내는 기능 추가 가능
