# 휴가 제도

## 연차 휴가

근속 기간에 따라 다음과 같이 부여됩니다.

| 근속 기간 | 부여 일수 |
|-----------|-----------|
| 1년 미만 | 매월 만근 시 1일 (최대 11일) |
| 1년 이상 ~ 3년 미만 | 15일 |
| 3년 이상 | 1년마다 1일 추가 (최대 25일) |

## 경조 휴가

| 구분 | 일수 |
|------|------|
| 본인 결혼 | 5일 |
| 자녀 결혼 | 1일 |
| 형제·자매 결혼 | 1일 |
| 배우자 출산 | 10일 |
| 본인·배우자 부모 사망 | 5일 |
| 본인·배우자 조부모 사망 | 3일 |
| 형제·자매 사망 | 3일 |

## 출산·육아 휴가

- **출산 전후 휴가**: 90일 (다태아 120일)
- **배우자 출산 휴가**: 10일
- **육아 휴직**: 자녀 1명당 최대 1년 (만 8세 이하 또는 초등학교 2학년 이하)

## 기타 휴가

- **병가**: 연 30일 이내 유급
- **공가**: 법적·공적 사유 발생 시
- **포상 휴가**: 회사 지정일에 부여

::: warning 휴가 신청
모든 휴가는 사전에 인트라넷을 통해 신청하고 결재를 받아야 합니다.
:::
