# 경조사 지원

대교홀딩스는 임직원의 경조사에 다음과 같이 지원합니다.

## 경조금

| 구분 | 금액 |
|------|------|
| 본인 결혼 | 500,000원 |
| 자녀 결혼 | 300,000원 |
| 본인·배우자 부모 회갑·고희 | 200,000원 |
| 본인·배우자 출산 | 300,000원 |
| 본인·배우자 부모 사망 | 500,000원 |
| 본인·배우자 조부모·형제자매 사망 | 200,000원 |

## 화환 및 조화

- 결혼: 회사 명의 화환 발송
- 부고: 회사 명의 조화 발송

## 경조 휴가

[휴가 제도](./leave#경조-휴가)를 참고해 주세요.

## 신청 방법

1. 사내 인트라넷 → 복리후생 → 경조사 신청
2. 증빙 서류 첨부 (청첩장, 사망진단서, 가족관계증명서 등)
3. 부서장 결재 후 경영전략팀 처리

::: tip 처리 기간
신청 후 영업일 기준 3일 이내에 지급됩니다. 긴급한 경우 경영전략팀(내선 0219)으로 직접 문의해 주세요.
:::
