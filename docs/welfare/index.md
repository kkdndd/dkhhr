# 복리후생 개요

대교홀딩스는 임직원과 가족의 삶의 질을 높이고 건강과 성장을 지원하기 위해 다양한 복리후생 제도를 운영합니다.

## 분류

- **[성장/개발](./growth)** — 어학비, 도서비, 직무교육, 자녀학자금 지원
- **[복지/여가](./leisure)** — 그룹사 제품 할인, 법인휴양소, 호텔 할인, 소셜크루, 생일/명절선물, 경조금, 장례금
- **[건강지원](./health)** — 단체상해보험, 건강검진

::: tip 신청 방법
대부분의 복리후생은 사내 인트라넷에서 신청할 수 있습니다. 신청이 어려우신 경우 경영전략팀(내선 0219)으로 문의해 주세요.
:::
