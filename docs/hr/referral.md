# 사내추천제도

대교홀딩스는 임직원이 우수 인재를 추천하는 사내추천제도를 운영하여 빠르고 적합한 인재 영입을 지원합니다.

## 제도 개요

| 항목 | 내용 |
|------|------|
| 추천자 | 전 임직원 (경영전략팀·임원 일부 제외) |
| 대상 직무 | 회사가 사전 공지한 채용 직무 |
| 추천 횟수 | 임직원 1인당 분기별 최대 3건 |
| 포상금 | 합격·입사 시 추천자에게 지급 |

## 추천 절차

1. **채용 공고 확인** — 인트라넷 "채용 → 사내추천 채용 공고"에서 모집 직무 확인
2. **후보자 추천** — 추천 양식 작성 후 경영전략팀에 제출
   - 후보자 이력서
   - 추천 사유 (300자 이상)
   - 추천자-후보자 관계
3. **경영전략팀 검토** — 1차 적합성 검토
4. **채용 절차 진행** — 일반 채용과 동일 절차 (서류 → 면접 → 합격)
5. **포상금 지급** — 입사 후 3개월 근무 시 추천자에게 지급

## 추천 포상금

| 채용 직급 | 포상금 |
|-----------|--------|
| 사원 | 100만원 |
| 대리 | 150만원 |
| 과장 | 200만원 |
| 차장 이상 | 300만원 |

::: tip 지급 방식
포상금은 입사자 3개월 근속 후 추천자 급여에 일괄 지급됩니다. 6개월 후에 동일 금액이 한 번 더 지급되어 총 2회로 나누어 지급됩니다.
:::

## 추천 시 주의사항

::: warning 추천 가능 후보자
- 본인이 직접 알고 있는 후보자만 추천 가능
- 후보자에게 사전 동의를 받고 추천해 주세요
- 동일 후보자를 여러 임직원이 중복 추천한 경우 최초 추천자에게 포상금 지급
:::

::: warning 추천 불가 대상
- 본인 직계 가족 (배우자, 부모, 자녀, 형제·자매)
- 최근 1년 이내 퇴사자
- 진행 중인 일반 채용에 이미 지원한 후보자
:::

## 추천 후 진행 상황 확인

- 추천 후 경영전략팀에서 진행 상황을 단계별로 추천자에게 안내
- 인트라넷 "채용 → 내가 추천한 후보" 메뉴에서도 확인 가능
