# 장기근속포상

대교홀딩스는 회사와 오랜 시간 함께한 임직원의 헌신에 감사하는 마음으로 장기근속포상을 운영합니다.

## 포상 체계

| 근속 연수 | 포상 내용 |
|-----------|-----------|
| **5년** | 표창장 + 부상 50만원 + 휴가 3일 |
| **10년** | 표창장 + 부상 200만원 + 휴가 5일 + 가족 호텔 숙박권 |
| **15년** | 표창장 + 부상 300만원 + 휴가 7일 |
| **20년** | 표창장 + 부상 500만원 + 휴가 10일 + 해외여행 지원 (1인 200만원 한도) |
| **25년** | 표창장 + 부상 700만원 + 휴가 14일 + 해외여행 지원 (1인 300만원 한도) |
| **30년** | 표창장 + 부상 1,000만원 + 휴가 20일 + 해외여행 지원 (1인 500만원 한도) |

## 시상 시기

- **정기 시상**: 매년 11월 창립기념일 행사
- **개별 시상**: 본인 입사 기념일 (선택 가능)

## 시상 절차

1. 인사팀이 연 단위로 장기근속 대상자 명단 작성
2. 대상자에게 포상 안내 메일 발송 (정기 시상 1개월 전)
3. 부상 및 휴가 신청
4. 시상식 참석 또는 개별 시상 진행

## 장기근속 휴가 사용

**[사용 가능 기간]**
- 시상 시점으로부터 **1년 이내**

**[사용 방법]**
- 인트라넷 "근태관리 → 휴가신청"에서 "장기근속휴가" 선택
- 연차와 별도로 부여되며, 분할 사용 가능

::: tip 안식 휴가와의 결합
연차와 결합하여 연속 사용이 가능합니다. 예를 들어 20년 근속 휴가 10일 + 연차 5일 = 15일 연속 휴가도 가능합니다.
:::

## 해외여행 지원 (20년 이상)

**[지원 내용]**
- 본인 + 동반 1인까지 항공·숙박·여행 경비 실비 지원
- 사용 기간: 시상 시점으로부터 **2년 이내**

**[신청 방법]**
1. 여행 계획 수립 후 인사팀에 사전 신고
2. 여행 종료 후 영수증 첨부하여 정산 신청
3. 익월 급여에 반영

::: warning 유의사항
- 동반자는 배우자 또는 직계가족(부모·자녀)에 한합니다.
- 여행 경비 한도 초과분은 본인 부담입니다.
:::

## 30년 이상 근속자 특별 예우

- 명예의 전당 등재 (본사 1층 게시)
- 평생 동문회 가입 자격 부여
- 회사 주요 행사 초청
