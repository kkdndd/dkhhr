# 평가 제도

## 평가 체계

대교홀딩스의 평가 체계는 **성과 평가**와 **역량 평가**로 구성됩니다.

### 1. 성과 평가
- **시기**: 연 2회 (상반기 7월, 하반기 1월)
- **방법**: KPI 기반 목표 달성도 평가
- **비중**: 70%

### 2. 역량 평가
- **시기**: 연 1회 (12월)
- **방법**: 다면 평가 (상사·동료·부하)
- **비중**: 30%

## 평가 등급

| 등급 | 비율 | 의미 |
|------|------|------|
| S | 상위 10% | 탁월 |
| A | 상위 30% | 우수 |
| B | 중간 50% | 양호 |
| C | 하위 10% | 보통 |

## 평가 결과 활용

평가 결과는 다음에 반영됩니다.

- 연봉 인상률
- 성과급 지급
- 승진 심사
- 교육 우선 순위

::: warning 이의 신청
평가 결과에 이의가 있을 경우 결과 통보 후 14일 이내에 인사팀에 신청할 수 있습니다.
:::
